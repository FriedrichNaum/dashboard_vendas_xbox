# Dashboard_vendas_xbox
Dashboard criada a partir de dados de assinaturas da Xbox. Ao receber tais perguntas de negocio, foram criadas tabelas dinamicas para analise dos dados para transformação em informações para posteriores analises.
Pergunta de Negócio 1 - Qual faturamento Total de vendas de planos anuais(contendo todas as assinaturas agregadas)
Pergunta de Negócio 2 - Qual faturamento Total de vendas de planos anuais, separado por auto renovação não é por auto renovação
Pergunta de Negocio 3 - Total de Vendas de Assinaturas do EA Play
Pergunta Negocio 4 - Total de Vendas de Assinaturas do Minecraft Season Pass

Os valores são relacionados a assinaturas e filtradas a partir de 3 informações, se a assinatura é mensal, em quatro meses ou anual.
